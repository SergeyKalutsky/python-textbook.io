# Типы данных

## Числовые данные
В **Python** два основных числовых типа данных:

**float** и **integer**

Целые числа представлены типом **integer( -2, 2, 3, 1, 0, ... )** , а дробные типом **float (0.2, 0.3, -0,22, 5,4 ...)**

Операции над числовыми данными, включают в себя:

1. Сложение (+)
```py
>>> print(2 + 3)
5
>>> print(2 + 2.5)
4.5
>>> print(0.2 + 0.3)
0.5
```
2. Вычитание (-)
```py
>>> print(2 - 3)
-1
>>> print(2 - 2.5)
-0.5
>>> print(0.2 - 0.3)
-0.099999999999
```
3. Умножение (*)
```py
>>> print(2 * 3)
6
>>> print(2 * 2.5)
5.0
>>> print(0.2 * 0.3)
0.06
```
4. Деление (/)
```py
>>> print(2 / 3)
0.6666666666666
>>> print(2 / 2.5)
0.8
>>> print(0.2 / 0.3)
0.66666666666667
```
5. Возведение в степень (**)
```py
>>> print(2 ** 3)
8
>>> print(2 ** 2.5)
5.656854249492381
>>> print(0.2 ** 0.3)
0.6170338627200097
```
6. Остаток от деления (%)
```py
>>> print(4 % 1)
0
>>> print(4 % 2)
0
>>> print(4 % 3)
1
>>> print(4 % 4)
0
```
7. Целочисленное деление (//)
```py
>>> print(6 // 1)
6
>>> print(6 // 2)
3
>>> print(6 // 4)
1
>>> print(6 // 5)
1
```
